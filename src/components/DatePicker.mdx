# Autocomplet

Formik is required to use DatePicker
@date-io/date-fns is required to use DatePicker
@material-ui/pickers is required to use DatePicker
date-fns is required to use DatePicker

Declare:
MuiPickersUtilsProvider with utils=DateFnsUtils

## Properties

<Props of={DatePicker} />

## Basic usage

<Playground>
    <MuiPickersUtilsProvider utils={DateFnsUtils}>
      <div style={{padding: 40}}>
        <Formik
            initialValues={{createdAt: new Date()}}
            onSubmit={(values) => console.log(values)}
            enableReinitialize
            >
              {({ isSubmitting }) => (
                <Form>
                  <DatePicker
                    name="createdAt"
                    label="Criado Em"
                  />
                </Form>
            )}
        </Formik>
      </div>
    </MuiPickersUtilsProvider>
</Playground>
